# FreeCAD

1. New document
2. Workbench = Part Design
3. Create new object
4. Create sketch
    - Choose plane
    - Choose polyline tool
      + z `M` spreminjaš način risanja
    - Kotiraš
      + `Shift+h` horizontalno
      + `Shift+v` navpično
5. Task -> Use 3D tools
    - Pad
    - Pocket
    - ...
6. Označiš ploskev na kateri želiš dodati element predmeta
7. Repeat from #4.
